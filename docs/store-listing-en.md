# Chrome Web Store / Edge Add-ons — English Listing

## Name

InterstitialBlocker

## Short Description (132 characters max)

Automatically dismisses intrusive interstitial ads so you can browse without interruption. No ad blocking — just overlay removal.

## Detailed Description

**Tired of full-screen interstitial ads interrupting your browsing?**

InterstitialBlocker detects and automatically dismisses interstitial advertisements — the full-page overlays that appear between page views and block your content. Unlike traditional ad blockers, it does not remove banner ads, video ads, or other non-intrusive advertising.

**How it works:**

InterstitialBlocker uses visual feature scoring to identify interstitial overlays in real time. It analyzes element positioning, size, z-index, and other visual properties — no blocklists or DOM pattern dictionaries needed.

When an interstitial is detected, the extension first tries to click its close button. If that fails, it safely removes the overlay so you can continue reading.

**Key features:**

- Blocks only interstitial ads — banners and other ads are left untouched
- Visual scoring algorithm — no filter lists to maintain or update
- Works on any website automatically
- Finds and clicks close buttons intelligently
- Per-site whitelist — easily disable on sites you trust
- Lightweight — minimal impact on page performance
- No data collection — all processing happens locally in your browser
- Works on Chrome and Edge

**What it does NOT do:**

- Does not block banner ads, sidebar ads, or video ads
- Does not collect or send any user data
- Does not modify page content beyond removing interstitial overlays
- Does not interfere with cookie consent dialogs, login modals, or age verification prompts

**Settings:**

- Global on/off toggle
- Per-site whitelist
- Adjustable detection sensitivity
- Detection history viewer

If you encounter a false positive (legitimate content being hidden) or a missed interstitial, please report it through the support page.
