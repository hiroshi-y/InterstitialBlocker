# Chrome Web Store / Edge Add-ons — 日本語掲載情報

## 名前

InterstitialBlocker

## 短い説明（132文字以内）

ページ間に表示される全画面インターステーシャル広告を自動で閉じます。広告ブロッカーではありません。オーバーレイの除去に特化。

## 詳細な説明

**ページを開くたびに表示される全画面広告にうんざりしていませんか？**

InterstitialBlockerは、インターステーシャル広告（ページ遷移時に表示される全画面オーバーレイ）を検出し、自動的に閉じるブラウザ拡張機能です。従来の広告ブロッカーとは異なり、バナー広告や動画広告などの邪魔にならない広告は除去しません。

**仕組み：**

InterstitialBlockerは、ビジュアル特徴スコアリングを使用してインターステーシャルをリアルタイムで識別します。要素のポジション、サイズ、z-index、その他の視覚的特性を分析し、ブロックリストやDOMパターン辞書は不要です。

インターステーシャルを検出すると、まず閉じるボタンのクリックを試みます。それが効果がない場合、安全にオーバーレイを除去して、コンテンツの閲覧を継続できるようにします。

**主な機能：**

- インターステーシャル広告のみをブロック — バナー等の他の広告はそのまま
- ビジュアルスコアリングアルゴリズム — フィルターリストの管理や更新が不要
- あらゆるウェブサイトで自動的に動作
- 閉じるボタンをインテリジェントに検出してクリック
- サイトごとのホワイトリスト — 信頼するサイトでは簡単に無効化
- 軽量 — ページパフォーマンスへの影響を最小限に
- データ収集なし — すべての処理はブラウザ内で完結
- ChromeとEdgeで動作

**やらないこと：**

- バナー広告、サイドバー広告、動画広告のブロックはしません
- ユーザーデータの収集・送信はしません
- インターステーシャル除去以外のページコンテンツの変更はしません
- Cookie同意ダイアログ、ログインモーダル、年齢確認画面への干渉はしません

**設定：**

- グローバルON/OFFトグル
- サイトごとのホワイトリスト
- 検出感度の調整
- 検出履歴の確認

誤検出（正当なコンテンツが非表示にされた場合）や見逃されたインターステーシャルがあれば、サポートページからご報告ください。
